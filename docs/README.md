# Библиотека геометрических фигур

## Обзор проекта

Данный проект представляет собой библиотеку на Python для вычисления площадей и периметров основных геометрических фигур. Реализация включает четыре фигуры с соответствующими модульными тестами, обеспечивающими высокую надежность кода.

## Поддерживаемые фигуры

### Круг
- **Площадь**: π × радиус²
- **Периметр**: 2 × π × радиус

### Треугольник
- **Площадь**: ½ × основание × высота
- **Периметр**: сторона₁ + сторона₂ + сторона₃

### Квадрат
- **Площадь**: сторона²
- **Периметр**: 4 × сторона

### Прямоугольник
- **Площадь**: длина × ширина
- **Периметр**: 2 × (длина + ширина)

## Детали реализации

### Операции с кругом
```python
import math

def area(r):
    """
    Вычисляет площадь круга.

    Аргументы:
        r (float): Радиус круга

    Возвращает:
        float: Площадь круга
    """
    return math.pi * r * r

def perimeter(r):
    """
    Вычисляет длину окружности.

    Аргументы:
        r (float): Радиус круга

    Возвращает:
        float: Длина окружности
    """
    return 2 * math.pi * r
```

**Примеры использования:**
```python
area(3)      # Возвращает 28.274333882308138
perimeter(2) # Возвращает 12.566370614359172
```

### Операции с треугольником
```python
def area(a, h):
    """
    Вычисляет площадь треугольника.

    Аргументы:
        a (float): Длина основания треугольника
        h (float): Высота треугольника

    Возвращает:
        float: Площадь треугольника
    """
    return 0.5 * a * h

def perimeter(a, b, c):
    """
    Вычисляет периметр треугольника.

    Аргументы:
        a, b, c (float): Три стороны треугольника

    Возвращает:
        float: Периметр треугольника
    """
    return a + b + c
```

**Примеры использования:**
```python
area(3, 4)       # Возвращает 6.0
perimeter(3,4,5) # Возвращает 12
```

### Операции с квадратом
```python
def area(a):
    """
    Вычисляет площадь квадрата.

    Аргументы:
        a (float): Длина стороны квадрата

    Возвращает:
        float: Площадь квадрата
    """
    return a * a

def perimeter(a):
    """
    Вычисляет периметр квадрата.

    Аргументы:
        a (float): Длина стороны квадрата

    Возвращает:
        float: Периметр квадрата
    """
    return 4 * a
```

**Примеры использования:**
```python
area(5)      # Возвращает 25
perimeter(3) # Возвращает 12
```

### Операции с прямоугольником
```python
def area(a, b):
    """
    Вычисляет площадь прямоугольника.

    Аргументы:
        a (float): Длина прямоугольника
        b (float): Ширина прямоугольника

    Возвращает:
        float: Площадь прямоугольника
    """
    return a * b

def perimeter(a, b):
    """
    Вычисляет периметр прямоугольника.

    Аргументы:
        a (float): Длина прямоугольника
        b (float): Ширина прямоугольника

    Возвращает:
        float: Периметр прямоугольника
    """
    return 2 * (a + b)
```

**Примеры использования:**
```python
area(4, 5)      # Возвращает 20
perimeter(3, 4) # Возвращает 14
```
